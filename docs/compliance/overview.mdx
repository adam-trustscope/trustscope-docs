# Compliance Overview

TrustScope helps organizations meet AI governance and compliance requirements across multiple frameworks.

## Supported Frameworks

| Framework | Coverage | Export Format |
|-----------|----------|---------------|
| **NIST AI RMF** | Full | NIST JSON |
| **EU AI Act** | Full | EU-compliant |
| **SOC 2** | Type II | Evidence pack |
| **HIPAA** | Technical safeguards | Audit logs |
| **OWASP Agentic** | Full | Security report |

## Key Capabilities

### Immutable Audit Trail
Every AI interaction is cryptographically hashed and linked:
- Tamper-evident logging
- Chain of custody
- Bitcoin anchoring (optional)

### Automated Evidence Collection
Generate compliance reports with one click:
- Risk assessments
- Control mappings
- Incident timelines

### Real-time Monitoring
Continuous compliance monitoring:
- Policy violation alerts
- Anomaly detection
- SLA tracking
