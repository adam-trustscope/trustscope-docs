# Node.js SDK Quickstart

Get started with the TrustScope Node.js SDK in 5 minutes.

## Installation

```bash
npm install @trustscope/sdk
# or
yarn add @trustscope/sdk
```

**Requirements:** Node.js 18+

---

## 1. Initialize

```typescript
import { TrustScope } from '@trustscope/sdk';

const ts = new TrustScope({
  apiKey: 'ts_live_xxx',  // Or set TRUSTSCOPE_API_KEY env var
  failOpen: true,          // Default: continue on errors
  timeout: 5000,           // Request timeout in ms
});
```

---

## 2. Observe Actions

```typescript
const result = await ts.observe({
  agentId: 'my-agent',
  actionType: 'chat_completion',
  parameters: { prompt: userMessage },
});

console.log(result.traceId);  // tr_abc123
```

---

## 3. Enforce Policies

Check policies **before** executing an action:

```typescript
import { Policy } from '@trustscope/sdk';

const decision = await ts.enforce({
  agentId: 'refund-bot',
  actionType: 'process_refund',
  parameters: { amount: 500, customer_id: 'cust_123' },
  policies: [Policy.maxValue('amount', 1000)],
});

if (decision.allowed) {
  // Proceed with refund
} else {
  console.log('Blocked by:', decision.blockedBy);
}
```

---

## 4. Use Sessions

Group related actions into sessions:

```typescript
const session = await ts.startSession('Customer Support');

await session.observe({
  agentId: 'my-agent',
  actionType: 'query',
  parameters: { q: 'order status' },
});

await session.observe({
  agentId: 'my-agent',
  actionType: 'response',
  parameters: { answer: '...' },
});

await session.end('completed');
```

## 5. Policy Builder

Build policies programmatically (11 factory methods):

```typescript
import { Policy } from '@trustscope/sdk';

// Monitor tier
Policy.maxValue('cost_usd', 100);
Policy.minValue('confidence_score', 0.8);
Policy.allowedValues('model', ['gpt-4', 'claude-3-opus']);

// Protect tier
Policy.blockedValues('action_type', ['delete']);
Policy.rateLimit(1000, { window: '1h' });
Policy.timeWindow('09:00', '17:00', { timezone: 'UTC' });
Policy.blockedDomains(['example.com']);

// Enforce tier
Policy.cumulativeLimit('tokens_used', 1000000, { window: '30d' });
Policy.humanApproval({ actionTypes: ['external_api'] });
Policy.chainDepthLimit(3);
Policy.externalCheck('https://approval.internal.com/check');
```

## Express Middleware

```typescript
import express from 'express';

const app = express();

app.use(async (req, res, next) => {
  const decision = await ts.enforce({
    agentId: 'api-agent',
    actionType: req.path,
    parameters: req.body,
  });

  if (!decision.allowed) {
    return res.status(403).json({ blocked: true, reason: decision.blockedBy });
  }

  next();
});
```
